# Fuel Tracker

A very small static app for estimating:

- trip distance
- fuel needed
- estimated fuel cost
- cost per mile

## Why this setup

This project uses plain `HTML`, `CSS`, and `JavaScript`, so it can be hosted directly on GitHub Pages without any build process.

## Local use

Open `index.html` in a browser.

## GitHub Pages

1. Push these files to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Set the source to deploy from your main branch.
4. Save, then GitHub Pages will publish the site.

## Future direction

The next logical enhancement is integrating a maps API so distance can be calculated automatically from an origin and destination.
